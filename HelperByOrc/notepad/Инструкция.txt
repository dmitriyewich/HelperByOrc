#bg15212b #coloreef7ff #pad16 Этот файл сам является живой демонстрацией разметки. Всё, что начинается с команды в начале строки, notepad.lua пытается отрисовать как форматирование.
#CoLoR7de2d1 Команды не чувствительны к регистру: #color, #COLOR и #CoLoR работают одинаково.

#font18 #colorfac51c 1. Как парсер читает строку
#indent24 Директивы распознаются только в начале строки, после необязательных пробелов.
#indent24 Можно ставить несколько директив подряд: цвет, фон, выравнивание, размер, иконку, отступы и так далее.
#indent24 Если строка начинается с обычного текста, то всё после него остаётся обычным текстом и не считается командой.
#indent24 Чтобы показать команду буквально, начинай строку не с решётки.
Пример буквального текста: #center #font18 #colorffaa00 Этот фрагмент не выполнится, потому что строка началась со слова "Пример".
Пример буквального цвета: {ff8800} этот фрагмент тоже не выполнится, потому что цветовой префикс не стоит в начале строки.

#font18 #colorfac51c 2. Цвет, фон и прозрачность
{ff6600}Это строка окрашена через префикс {RRGGBB} в начале строки.
#color00d084 Это строка окрашена через #colorRRGGBB.
#bg203040 #colorffffff Строка с фоном через #bgRRGGBB и белым текстом поверх.
#bg203040 #colorffffff #alpha60 Тот же принцип, но уже с прозрачностью через #alpha60.
#bg3a1f1f #colorffd6d6 #alpha35 Чем меньше значение alpha, тем прозрачнее строка. Диапазон: от 0 до 100.
#indent24 Цвет в фигурных скобках и #color работают только как префиксы строки.
#indent24 #alpha влияет и на текст, и на фон, и на иконки, и на изображения в этой строке.

#font18 #colorfac51c 3. Выравнивание и #sameline
#left Это обычное выравнивание влево. Команда #left нужна редко, но поддерживается.
#center #color7de2d1 Строка по центру.
#right #color7de2d1 Строка по правому краю.
Левая часть общей строки
#sameline #right #colorffd166 Правая часть этой же строки через #sameline + #right
#indent24 #sameline приклеивает текущую строку к предыдущей логической строке.
#indent24 Чаще всего #sameline используют вместе с #right, чтобы получить левую и правую часть в одной строке.

#font18 #colorfac51c 4. Отступы
#indent24 Отступ через #indent24.
#pad40 Тот же приём через #pad40. Это синоним #indent.
#tab Один tab без числа равен 32 пикселям.
#tab2 Два tab дают 64 пикселя.
#indent8 Отступы суммируются, если поставить несколько команд подряд.
Синтаксис: #indent-16 Отрицательный отступ тоже поддерживается, но использовать его нужно осторожно.

#font18 #colorfac51c 5. Размер шрифта
#font12 Размер 12
#font14 Размер 14
#font16 Размер 16
#font18 Размер 18
#font30 Размер 30
#indent24 Поддерживаются только размеры: 12, 14, 16, 18 и 30.
Пример обычного текста: #font20 Эта команда не сработает как форматирование и останется простым текстом.

#font18 #colorfac51c 6. Преобразование регистра
#upper эта строка станет верхним регистром.
#lower ЭТА СТРОКА СТАНЕТ НИЖНИМ РЕГИСТРОМ.
#indent24 Преобразование применяется к текстовой части строки.

#font18 #colorfac51c 7. Списки и иконки
#bullet Обычный пункт списка через #bullet.
#bullet Второй пункт списка.
#iconSTAR #bullet Можно совмещать иконку и bullet в одной строке.
#iconBOOK Иконка без списка тоже работает.
#iconCOMPASS #color9fd0ff Иконка берётся по имени из Font Awesome 7.
#indent24 Синтаксис иконки: #iconИМЯ_ИКОНКИ
#indent24 Если имя не найдено, будет показан текстовый плейсхолдер вида [ICON_NAME].

#font18 #colorfac51c 8. Пустые строки и разделители
Строка до #br.
#br
Строка после одного дополнительного пустого интервала.
Строка до #br2.
#br2
Строка после двух дополнительных пустых строк.
#hr
#hrff8800
#indent24 #br без числа добавляет одну пустую строку, #br2 добавляет две, и так далее.
#indent24 #hr лучше ставить отдельной строкой без текста. Цветной вариант: #hrRRGGBB

#font18 #colorfac51c 9. Картинки через #img
#img(test.png)
#sameline #pad16 #font14 #colorcfe8ff Локальная картинка из папки HelperByOrc\notepad\images\test.png
#img(test.png, size(156,124))
#sameline #pad16 #font14 #colorffd27a Пример фиксированного размера size(156,124)
#img(test.png, size(220,-1))
#sameline #pad16 #font14 #color9fe5c1 Ширина задана, высота посчитана автоматически: size(220,-1)
#img(test.png, size(-1,96))
#sameline #pad16 #font14 #color9fe5c1 Высота задана, ширина посчитана автоматически: size(-1,96)
#img(https://s.fotora.ru/5067870210cae2be.png, size(156,124))
#sameline #pad16 #font14 #colorffd27a Пример загрузки по URL через #img(https://...)
#indent24 Относительный путь ищется внутри папки images. Для этого файла достаточно написать просто test.png.
#indent24 Аргументы size(...) и pos(...) можно писать в любом порядке.
#indent24 Если использовать pos(x,y), картинка рисуется по абсолютной позиции внутри окна и не занимает место в потоке текста.
#indent24 Абсолютное позиционирование может перекрыть текст ниже, поэтому его лучше применять осознанно.
#indent24 Если файл не найден или URL ещё не скачался, строка зарезервирует место, но сама картинка может быть пустой до загрузки.
Синтаксис локального файла: #img(test.png)
Синтаксис URL-изображения: #img(https://s.fotora.ru/5067870210cae2be.png)
Синтаксис: #img(test.png)
Синтаксис: #img(test.png, size(220,140))
Синтаксис: #img(test.png, size(220,-1))
Синтаксис: #img(test.png, size(-1,96))
Синтаксис: #img(test.png, size(-1,-1))
Синтаксис: #img(test.png, pos(40,120))
Синтаксис: #img(test.png, pos(40,120), size(220,140))
Синтаксис: #img(weapons/test.png)
Синтаксис: #img(C:\Images\test.png)
Синтаксис: #img(https://s.fotora.ru/5067870210cae2be.png)
#indent24 URL-изображения кешируются в папку images\url\

#font18 #colorfac51c 10. Переменные в фигурных скобках
#bg13202b #colore7f3ff #pad12 В предпросмотре работают теги из модуля Tags. Например: {myorg}
#sameline #right #colorffd27a И ещё один пример: {myorgrang}
#indent24 Если в Tags созданы свои переменные, их тоже можно подставлять в тексте этой инструкции и в обычных заметках.
#indent24 Исключение: конструкция вида {RRGGBB} в самом начале строки трактуется как цвет, а не как переменная.

#font18 #colorfac51c 12. Практические подсказки
#bullet Для заголовков удобно сочетание: #center + #font18 или #font30 + #color...
#bullet Для аккуратных блоков-подсказок удобно сочетание: #bg... + #color... + #pad...
#bullet Для двух колонок в одной строке удобно: обычная строка, а следующая строка начинается с #sameline #right
#bullet Для списков хватает #bullet, а если нужен более заметный акцент, добавляй ещё и #iconSTAR или #iconBOOK
#bullet Для изображений по умолчанию лучше использовать относительный путь внутри images, чтобы файл оставался переносимым
