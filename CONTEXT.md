# CONTEXT.md

## Среда
- Lua 5.1 (LuaJIT 2.1.0-beta3), MoonLoader (SA-MP)
- UI: mimgui (Dear ImGui 1.72). Использовать только API mimgui 1.72
- Можно FFI и bit, но только когда реально нужно

## Hot reload (обязательно)
Изменения в src/ или lib/ могут перезагрузить проект. Код обязан переживать reload:
- отписка от событий/коллбеков
- закрытие окон, освобождение ресурсов (рендер, текстуры, файлы и т.п.)
- не плодить корутины/таймеры/хуки, не оставлять старые висеть

## require и зависимости
- src: код проекта
- lib: только библиотеки и переиспользуемые модули
- require сначала видит локальный lib/, потом стандартные пути MoonLoader
- без luarocks/npm, только то, что реально доступно в окружении MoonLoader/moonly

## Сборка/экспорт
- сборка делается инструментами moonly вручную пользователем
- нельзя запускать команды moonly сборки из ассистента (включая moonly bundle)
- при упаковке/выгрузке следить за кодировкой (часто нужно UTF-8)

## Критические правила
### Про знак %
* функции, которые используют printf-формат.
- в Text/TextWrapped/TextColored/BulletText и похожих:
  - плохо: "%s"
  - хорошо: "%%s"
- любой текст, где может быть %, обязан экранировать %

### Язык UI
- весь UI-текст только на русском

## Правила правок кода
- не редактировать существующие библиотеки в lib/ (только использовать)
- совместимость с Lua 5.1 / LuaJIT (без обязательного синтаксиса Lua 5.2+)
- не выдумывать API MoonLoader/mimgui, использовать только существующее
- git не использовать: проект локальный, без опоры на git-команды/историю
- не ломать публичные интерфейсы модулей
  - если меняешь сигнатуру, описать что поменялось и где править вызовы

## Как предлагать изменения
- рефакторинг: diff или точные фрагменты
- новый файл: точный путь (обычно src/...)
- если сомневаешься по API mimgui: сверяться с lib/mimgui/cdefs.lua
- Font Awesome: lib/fAwesome7.lua, таблица fa_icon

## Локальная проверка (коротко)
- для синтаксиса использовать luajit (например через loadfile), не luac -p
- финальная проверка всегда в рантайме MoonLoader/LuaJIT с учетом hot reload
